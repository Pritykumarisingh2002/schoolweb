This is a school management system that has two intreface, once for adminpanel and another for visitors. The visitor's website is accessible by everyone where visitors can access information about school, pay student's monthly fee, see studen's result, exam notifications and schedule, admission, holidays, upcoming events, important notifications, send feedback about teachers and suggestion about school where as adminpanel website is only  accessible by school's administration from where school can upload details about student, upload admission notification, track student fees, upload academic calender, booklist, student's results, exam schedule, upcoming events and many more things.
